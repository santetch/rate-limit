# System Design Challenge: Rate Limiter

This project contains an implementation of a Rate Limiter as part of a system design challenge.

## Contents
- `rate-limiter/`: The core implementation of the rate limiter.
  - `src/`: Source code (TypeScript).
  - `tests/`: Unit tests.
  - `DESIGN.md`: Detailed explanation of the architecture and trade-offs.

## Quick Start
```bash
cd rate-limiter
yarn install
yarn jest
```

## Chosen Problem
Design a Rate Limiter (Alex Xu, System Design Interview Vol 1).
Algorithm: Token Bucket.
Implementation: TypeScript.
